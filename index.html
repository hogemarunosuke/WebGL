遊べるゲームを作りたい

全てのゲームはWebGLです。PC版 Mozilla Firefox 及び Google Chrome にて動作確認済みです。

開発ブログ | GitHub | Twitter

2017-01-23 PrimitiveIsland

迫りくる赤八角柱を破壊し続けるゲームです。

<操作方法>
WASDキーで移動。スペースキーでジャンプ。マウスで視点移動。
左クリックで射撃。右クリックでロックオン。左シフトでダッシュ（前進中のみ）。

2017-01-03 PronamaChanShooting

プロ生ちゃんを操作して遊ぶシューティングゲームです。

<操作方法>
ショット：左クリックor左Ctrl，ボム：右クリックor左Alt。
移動：WASDorカーソルキー，ポーズ：スペース。

2016-10-24 KingOfTheHill

迫りくる赤玉を破壊し続けるゲームです。

<操作方法>
WASDキーで移動。スペースキーでジャンプ。マウスで視点移動。
左クリックで射撃。赤玉を倒して加点、触れるとゲームオーバー。

2016-10-17 SkyJumper

ひたすら次の足場へジャンプするゲームです。

<操作方法>
左クリックでジャンプ。ジャンプ中にWASDキーで移動。
次の足場に着地で得点。ジャンプは連続３回まで。

2016-10-11 CameraBox

タイミングに合わせてプロ生ちゃんを撮影するゲームです。

<操作方法>
青パネルが上下に自動移動。赤に重なるときにスペースor左クリック。
時間経過、空打ちで取得スコア減少。カーソル左右でアングル変更。

2016-10-07 AstroShooter

ひたすら隕石を破壊するゲームです。

<操作方法>
カーソルキー左右で移動、スペースでレーザー発射。
岩破壊で＋１ポイント、見逃しで－１ポイント。制限時間６０秒。

2016-10-02 BounceBall

ひたすら弾を跳ね返すゲームです。

<操作方法>
カーソルキー左右でバー移動。
